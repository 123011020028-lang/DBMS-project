# DBMS-project
🩸 Blood Donation Camp Coordination System
📌 Overview

The Blood Donation Camp Coordination System is a web-based application designed to efficiently manage and coordinate blood donation camps. It helps organizers, donors, and hospitals streamline the process of donor registration, camp scheduling, and blood inventory management.

This system aims to reduce manual effort and improve communication between all stakeholders involved in blood donation activities.

🚀 Features
👤 Donor Registration & Management
Register new donors
Maintain donor details (blood group, contact, history)
🏥 Camp Management
Create and schedule blood donation camps
Assign locations and organizers
📅 Appointment Scheduling
Donors can book slots for donation
Avoid overcrowding
🩸 Blood Inventory Tracking
Monitor available blood units
Track usage and shortages
🔔 Notifications & Alerts
Reminders for upcoming camps
Emergency blood requests
🔍 Search & Filter
Find donors by blood group and location
